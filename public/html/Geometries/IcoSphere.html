参数说明(options):
参数名 意义 默认值
points 构成线段的点数组(vec3[]) null
updatable 是否可更新(boolean) false
colors 颜色数组(Color4[]) null
useVertexColor 是否使用顶点颜色(boolean) false
注意事项:
1.Color4的第四个分量(alpha通道值)可以不填,默认是1.0;
2.圆环可以看作由n条线段组成的;
